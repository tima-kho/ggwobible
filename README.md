# Bible Presenter

Лёгкое fullstack-веб-приложение для церкви: вывод стихов из Библии на проектор / второй монитор. Включает полную Библию (66 книг) в двух переводах:

- **РСТ** — Русский Синодальный (1876, public domain)
- **KJV** — King James Version (1611, public domain)

## Возможности

- **Поиск** по тексту во всех ~31 000 стихах + по ссылке (`Иоанна 3:16`, `Псалом 22:1-3`).
- **Навигация** по 66 книгам → главам → стихам.
- **Предпросмотр** в админке с пропорциями 16:9, как на проекторе.
- **Экран проектора** в отдельном окне: огромный адаптивный шрифт, плавные fade-переходы, синхронизация через `localStorage`.
- **Режим «поверх всех окон»** через [Document Picture-in-Picture API](https://developer.chrome.com/docs/web-platform/document-picture-in-picture/) (Chrome/Edge 116+) — окно с текстом всегда поверх остальных.
- **Двуязычный режим** — переключение РСТ ↔ KJV в один клик.

## Архитектура

```
bible-presenter/
├── package.json
├── server/
│   ├── server.js          # Fastify HTTP-сервер
│   ├── library.js         # in-memory загрузка + поиск
│   └── books-meta.js      # справочник 66 книг (RU/EN/abbr)
├── scripts/
│   └── build-data.js      # скачивает Библии и режет на 66 файлов
├── data/                  # генерируется build-data.js
│   ├── books.json         # сводный индекс
│   ├── rst/               # 66 файлов: 01-genesis.json, 02-exodus.json, ...
│   └── kjv/               # 66 файлов
└── public/
    ├── index.html
    ├── styles.css
    └── app.js             # admin + screen в одном SPA
```

Каждый файл книги имеет вложенную структуру `chapters[].verses[]` — это и есть «файловая структура 66 книг с подразделением на главы и стихи».

## Установка и запуск

Требования: **Node.js 18+** и интернет (только при первой сборке данных).

```bash
npm install
npm run build:data    # скачает RST + KJV (~13 МБ), разложит по 132 файлам
npm start             # запустит сервер на http://127.0.0.1:3000
```

После первого `build:data` интернет больше не нужен — приложение работает полностью офлайн.

Для разработки с авторестартом:
```bash
npm run dev
```

## Использование

1. Откройте `http://127.0.0.1:3000` — это **панель управления**.
2. Сверху — переключатель переводов (РСТ / KJV).
3. Два таба:
   - **Поиск** — введите слово (`любовь`) или ссылку (`Иоанна 3:16`, `Псалом 117:24`).
   - **Книги** — все 66 книг → выберите → выберите главу → кликните на стих.
4. Открыть проектор одним из двух способов:
   - **«Открыть экран ТВ»** — обычное окно. Перетащите на проектор, кликните в нём (или нажмите F11) → перейдёт в полноэкранный режим, на втором мониторе ничего его не перекроет.
   - **«Поверх окон»** *(Chrome/Edge 116+)* — плавающее окно через Document Picture-in-Picture, которое ОС держит **всегда поверх** остальных окон. Удобно если работаете на одном мониторе или хотите, чтобы текст не пропал за свернутыми окнами.
5. Кликайте по стихам в админке — они мгновенно появляются на проекторе с плавной анимацией.
6. **«Очистить экран»** — плавно убирает текст, оставляя чёрный экран.

## REST API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/translations` | Список переводов |
| GET | `/api/books` | Индекс 66 книг |
| GET | `/api/books/:id?translation=rst` | Полная книга со всеми главами |
| GET | `/api/books/:id/chapters/:n?translation=rst` | Одна глава |
| GET | `/api/search?q=...&translation=rst&limit=80` | Поиск (по ссылке или тексту) |
| GET | `/healthz` | Проверка живости |

Параметр `translation` принимает `rst` или `kjv` (по умолчанию `rst`).
